¿Qué es Electrocomp?

Electrocomp es un sitio web desarrollado como parte del proyecto académico del curso de Diseño Web. Su objetivo es presentar de forma clara y profesional los servicios, productos y contenido informativo de un emprendimiento tecnológico.

El sitio busca ofrecer una experiencia moderna, rápida y accesible, siguiendo el enfoque Mobile First y aplicando buenas prácticas de diseño y usabilidad.

Tecnologías Utilizadas

HTML5

Estructura semántica del sitio con etiquetas modernas.

CSS3 & Bootstrap

Diseño responsivo, estilos visuales y componentes preparados.

JavaScript

Interactividad, animaciones y carga dinámica de contenido JSON.

jQuery

Efectos visuales y funciones rápidas para manipulación del DOM.

Google Maps API

Implementación de geolocalización para ubicar el negocio.

Multimedia HTML5

Reproducción de audio y video en las secciones correspondientes.
